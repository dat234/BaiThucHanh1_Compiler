## Hướng dẫn biên dịch và chạy chương trình

### 1. Biên dịch (Compile)
Mở Terminal tại thư mục bài tập và chạy lệnh sau để tạo file chạy:

gcc main.c -o indexer

.\indexer.exe vanban.txt

Hoặc

gcc main.c -o indexer

.\indexer.exe alice30.txt > output.txt
